Lilia Documentation

Welcome to the documentation for Lilia.

This is a basic skeleton that serves as a raw foundation for building any schema according to a player's preferences. It provides a starting point with essential functionalities and can be expanded upon to create custom gameplay experiences.

To utilize this skeleton, ensure you have the Lilia framework installed.

Developers

The sidebar displays the entire contents of the documentation for this schema. They are all searchable with the search box at the top of the sidebar.

Server Owners

If you're looking to get your server up and running as soon as possible, check out the Installation Tutorial guide in the manual. Remember that this schema is dependent on the Lilia framework.
