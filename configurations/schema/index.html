Schema

Schema configuration options.

This is meant to define the Schema's default information.

Tables

SchemaInfo

A list of schema fields that define the gamemode you are making. These options are defined by the 'SCHEMA' table.

Fields

  • name

    Controls the name of the schema | string.

  • author

    Specifies the author of the schema | string.

  • desc

    Defines the description of the schema | string.
